## Usage

- clone this repo
- set current directory to ```this_repo/predict_app```
- run ```docker build -t bitcoin:0 .```
- run ```docker run --name bitcoin -d -p 8080:8080 bitcoin:0```
- open your browser and type ```localhost:8080```
